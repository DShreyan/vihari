First open terminal in directory where package.json is located(Project folder)

add "start": "nodemon app.js", in package.json "scripts" part

Then install :
npm i
npm i express
npm i ejs
npm i body-parser
npm i --save-dev nodemon
npm i sqlite3

Then type npm start

open localhost:5000 in your browser
